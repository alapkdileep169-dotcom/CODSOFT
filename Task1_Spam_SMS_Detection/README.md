# Spam SMS Detection using Machine Learning

## Project Description
This project builds a machine learning model that classifies SMS messages as **Spam** or **Ham (Not Spam)** using natural language processing techniques.

## Algorithm Used
Naive Bayes Classifier

## Libraries Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer

## Dataset
SMS Spam Collection Dataset containing labeled SMS messages.

## Workflow
1. Load the SMS dataset
2. Preprocess the text messages
3. Convert text into numerical vectors using TF-IDF
4. Train the Naive Bayes classifier
5. Predict whether a message is spam or legitimate

## Output
The model predicts whether a message is **Spam** or **Not Spam**.
