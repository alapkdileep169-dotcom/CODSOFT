
# Customer Churn Prediction using Machine Learning

## Project Description
This project builds a machine learning model to predict whether a customer will leave a service (churn) or continue using it.

## Algorithm Used
Logistic Regression

## Libraries Used
- Python
- Pandas
- Scikit-learn

## Dataset
Dataset available at:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Workflow
1. Load customer dataset
2. Separate features and target variable
3. Train Logistic Regression model
4. Predict customer churn
5. Evaluate model accuracy
